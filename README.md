# Nafs Guard

A productivity and digital wellbeing app to block YouTube Shorts, Facebook Reels, unwanted content, set app usage limits, track screen time, and prevent impulsive unblocking with emergency challenge protection.

[View Project](https://butterflydevs.web.app/projects/nafsguard)

## Features

### 🚫 App Blocking
- Block any unwanted apps
- Schedule blocking for specific times
- Block bundles of apps together
- Emergency unlock challenge to prevent impulsive unblocking

### 📊 Screen Time Tracking
- Track daily screen time
- Weekly summary reports
- App usage statistics
- Usage limit notifications

### 🎯 Content Blocking
- YouTube Shorts blocking
- Facebook Reels blocking
- In-app content protection

### 🔒 Security Features
- Emergency unlock challenge
- Device admin protection
- Uninstall protection
- Accidental uninstall prevention

### 🎨 UI/UX
- Clean, modern interface
- Material Design 3
- Poppins fonts
- Beautiful color scheme
- Responsive design
